# Todo

Accepted proposals, ready to be detailed or implemented.

When the change needs a separate design pass, expand the doc and move it to
**`../plan/`**. When you start implementation, move the file to **`../work/`**
(and add a **Status log** section).

Small items may go **`todo/` → `work/`** directly if the proposal already contains
enough step-by-step detail (see `../README.md`).
