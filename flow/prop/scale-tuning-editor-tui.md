# In-TUI scale / tuning editor

## What

Edit `Scale` / `Tuning` parameters interactively: EDO size, root, maybe import Scala `.scl` (speculative), with live retuning of the pattern roll.

## Why

Xenharmonic pitch is a core thesis; hiding it behind code examples undersells the project.

## Notes

- Start with presets + a few numeric fields; full `.scl` is a larger crate boundary decision.
- Ensure pattern roll labeling updates with tuning name.
