# TUI accessibility: themes, contrast, motion

## What

High-contrast palette preset, optional reduced flicker (lower scope refresh rate), configurable color overrides via env or config file, documented terminal recommendations.

## Why

Spectrum/waveform panes and rapid updates can be hard on eyes; some users need WCAG-minded contrast in the terminal.

## Notes

- ratatui styling is limited; focus on palettes and refresh cadence first.
- Document terminal emulator settings (font size, bold as color).
