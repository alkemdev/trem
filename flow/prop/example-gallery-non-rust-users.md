# Example gallery for non-Rust users

## What

Curated `assets/` or `examples/` README entries: "import this MIDI, run this one command, hear X" with expected sound described in plain language.

## Why

Rust literacy should not be required to evaluate trem as a musical tool; lowers the intimidation factor.

## Notes

- Pair each example with a 10-second expected outcome (what to listen for).
- Link to `docs/install.md` for platform audio setup.
